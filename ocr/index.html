OCR

ตัวอย่างการทำ OCR บน เบราว์เซอร์ด้วย tesseract.js สามารถใช้ JavaScript ตรงๆโดยไม่ต้องใช้ Python เหมือนแบบอื่นๆ และมีความแม่นยำสูง ข้อดีคือไม่โหลดเซิร์ฟเวอร์ ข้อเสียคือไฟล์โมเดลแบบ best ขนาดเกิน 10MB ตอนนี้มีโปรแกรม demo สำหรับการ OCR หนังสือเดินทางกับเอกสารทั่วไป จะใช้โมเดลแบบต่างๆในการทดสอบ

โค้ดเขียนแบบง่ายๆสำหรับการนำเสนอ ก่อนใช้จริงควรมีการปรับปรุงให้ดียิ่งขึ้น

MRZ

จากการทดสอบ OCR หนังสือเดินทางของโปรเจ็ก JOBS ไม่ค่อยแม่นยำผิดพลาดเกือบทุกจุด ปัญหาคือ ต้นฉบับก็ไม่ค่อยชัด ถึงแม้เอกสารจะชัดเจนก็ตาม แต่ละประเทศมีรูปแบบพื้นหลังและฟอนต์ต่างกันไป ถ้าจะให้แม่นยำเหมือนเครื่องอ่านที่สนามบินต้องอ่านจาก Machine Readable Zone (MRZ)

โปรแกรมตัวอย่าง

ตัวอย่างนี้ใช้โมเดลที่เทรนมาสำหรับอ่าน MRZ โดยเฉพาะ เมื่อ OCR ได้ข้อความจะไปผ่าน parser ได้ JSON ออกมาใช้งาน ส่วน parser ยังต้องปรับปรุงให้ดีกว่านี้ ยังไม่ได้ทดสอบกับหนังสือเดินทางของไทยและเพื่อนบ้าน

โปรแกรมตัวอย่าง

OCR ภาพแปลงเป็น PDF แบบมีข้อความบนภาพ ใช้ โมเดลภาษามาตรฐาน ของ tesseract ควรเลือกใช้โมเดลให้ตรงกับภาษา ภาพด้านล่างเป็นตัวอย่าง OCR ภาษาพม่าผสมอังกฤษ แล้วแปลงเป็น PDF แบบมี text layer ทำการก็อปปี้ไปใส่ใน VS Code
